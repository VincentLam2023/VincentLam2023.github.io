This Power BI project analyzes human cognitive performance based on lifestyle factors such as sleep duration, stress level, caffeine intake, screen time, and more. Using two integrated datasets—human_cognitive_performance.csv and StudentPerformanceFactors.csv—the project identifies key trends and correlations, builds predictive models, and reveals hidden patterns through clustering. The goal is to inform health and productivity decisions by visualizing and predicting how daily habits affect mental sharpness. Key highlights include a clear negative impact of screen time on cognitive scores, strong predictive accuracy for memory performance using a Random Forest model, and segmentation of users based on lifestyle clusters. The application incorporates Python visualizations and machine learning models, including one developed in Azure ML.
